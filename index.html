Trigger Definitions
Format <a class="w3s-trigger/mover ..." name="" target="" title="" href="" rev="" rel="">Trigger</a>
AttributeDescription
name
Name Definition
ValueDescription
popupPopup in the middle of window, mostly with modal.
overlapBox which overlays on the specific target DOM.
floatFloat box which floats beside the specific target DOM.
actionAjax with JSON response.
loadAjax loading into target DOM.
reloadReloading target box or closest box which original content is ajax loading.
cloneClone current line which has class w3s-clone.
closeClose current popup or loverlap box.
resetReset closest or target form.
submitSubmit closeast or target form.
triggerTrigger click event on target DOM.
callbackCall target function with data in rev (separated by ',').
classtrigger:triggered by click event, mover:triggered by mouseover event.
class (optional)modal for popup only, inline float comments box
target (optional)Target DOM for ajax loading
title (optional)Title for popup or overlap, float
href (optional)URL for ajax, Dom ID (e.g. #id) for static info box (mouseover float box), or _token value for ajax submit (POST or GET).
rel (optional)Confirmation text. Mostly used for name is action. e.g. rel="Delete it?"
rev (optional)Additional param field ids for post when name is "action", e.g. rev="p1,p2,..."

Ajax JSON Response Defintion
Format{success:value, message:value, ...}
ParameterDescription
success
(mandatory)
(bool) true/falseprocessed OK or Failure
(string) closeClose current W3S box (popup, overlap or float)
(string) resetReset the form in current W3S box
(string) reloadReload page or W3S box specified (res.target).
(string) reset&reloadReset the form in current W3S box and reload W3S box specified (res.target)
(string) close&reloadClose current W3S box and reload W3S box which is specified in target
(string) triggerTrigger the W3S trigger (res.target) with specified type (res.action)
(string) close&triggerClose current W3S box and trigger the W3S trigger (res.target) with specified type (res.action)
(string) loadLoad res.url into DOM specified (res.target)
(string) close&loadClose current W3S box and load res.url into DOM specified (res.target)
(string) callCall local function which is specified target (res.target)
message(optional)The message will be popupped in alert window.
target(optional)DOM Id for operation.
url(optional)URL for ajax loading.
data(optional)Parameters (seperated by ',') for calling function.
action(optional)For "trigger" success. The action name.
